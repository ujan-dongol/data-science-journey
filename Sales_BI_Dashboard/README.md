# Sales BI Dashboard

A modular Business Intelligence (BI) dashboard built with Python to analyze sales data, calculate KPIs, and generate professional visual reports using Matplotlib.

## Project Overview

The Sales BI Dashboard project demonstrates end-to-end data analysis including:

- Data loading
- Data transformation
- KPI calculation
- Visual dashboard generation
- Report export
This project follows a clean modular architecture and mimics real-world BI systems.
## Features
 Clean modular structure (src package)
- KPI calculations
- Monthly revenue trend analysis
- Product performance breakdown
- Regional performance comparison
- Revenue distribution analysis
- Professional Matplotlib dashboard
- Exportable dashboard image
- Portfolio-ready structure

## Installation
- Clone Repository
git clone https://github.com/yourusername/Sales_BI_Dashboard.git
cd Sales_BI_Dashboard

- Create Virtual Environment (Recommended)
python -m venv venv
venv\Scripts\activate

- Install Dependencies
pip install -r requirements.txt

# How To Run
From project root directory:
- python main.py
or 
- streamlit run app.py
